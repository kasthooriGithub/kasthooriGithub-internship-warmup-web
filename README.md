# Archers Tek Website

This is a simple web application for Archers Tek, created as part of a web development mini project.

## Features

- Company Introduction page with:
  - Navigation bar
  - Banner section
  - About Us section with mission, vision, and values
- Contact Us page with:
  - Contact form (frontend only)
  - Contact information cards

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome icons

## Setup

1. Clone the repository
2. Open `index.html` in your browser for the Company Introduction page
3. Open `contact.html` in your browser for the Contact Us page

## Note

This is a frontend-only project with no backend functionality.
